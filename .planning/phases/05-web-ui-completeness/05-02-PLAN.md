## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Container.labels (wire) → Row.svelte gating | Server-side ACT-09/SAFE-01/SAFE-02 enforces label safety regardless of UI; the UI's `allowUpdate` check is UX-only. A malicious or stale client cannot bypass server enforcement. |
| Clipboard API | navigator.clipboard.writeText is gated by browser permissions; on a LAN-served `http://` URL Chrome 76+ permits it without prompt for the active document. Phase 1 + 5 test stack uses http://localhost:8080. |
| Tygo-generated types | The Container type in `ui/src/lib/types.d.ts` is the only source-of-truth wire shape; manual additions in plans would break `make check-types` (Phase 1 invariant). Plan 05-02 reads, never writes. |
