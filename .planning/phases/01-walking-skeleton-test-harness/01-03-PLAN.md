| build-time -> runtime | The embedded bundle is build-time only; no user-controlled HTML can be injected into the response |

|-----------|----------|-----------|-------------|-----------------|

| T-01-03-02 | Tampering | XSS via embedded bundle | accept | Vite bundle is build-time only; Svelte 5 escapes interpolated values by default; no user-controlled HTML in v1 |
| T-01-03-03 | Information Disclosure | source maps leak business logic | accept | Phase 1 dev-grade build; Phase 7 can disable sourcemaps via vite build.sourcemap: false if production hardening requires it |
| T-01-03-04 | Tampering | npm install supply-chain | mitigate | package-lock.json committed; CI uses `npm ci` (not `npm install`) so lockfile is enforced |
</threat_model>
