## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP request → compose subprocess argv | The service-name path parameter is the only operator-controllable input that ever flows into `exec.CommandContext`. Argv separation in runner.UpdateService (defense in depth with Plan 04-03 regex middleware) is the security-critical invariant. |
| compose subprocess → slog stdout | Compose stderr is captured into the error wrap chain; truncated copy goes to slog. Compose talks to the local daemon — no registry auth in stderr — so no Bearer/Basic redaction issue. The Phase-3 newRedactingHandler is still belt-and-braces. |
| Go process → host docker CLI | `exec.LookPath("docker")` at boot. ENOENT fails fast (no fallback). The host's compose v2 plugin is the version under test; we don't ship it. |
