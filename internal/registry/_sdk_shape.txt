# type names while source uses pkg.X form; the index below closes the

# crane.WithContext                -> github.com/google/go-containerregistry/pkg/crane.WithContext

# authn.Anonymous                  -> github.com/google/go-containerregistry/pkg/authn.Anonymous (var, singleton)
# v1.Platform                      -> github.com/google/go-containerregistry/pkg/v1.Platform (struct)
#
# FORBIDDEN identifier (Pitfall 2 — Centroid CONTEXT.md Area 1):
#   The package-qualified identifier (constructed as the literal three
#   tokens authn / dot / DefaultKeychain) maps to
#   github.com/google/go-containerregistry/pkg/authn DefaultKeychain.
#   This identifier is BANNED in internal/registry/. The CI grep guard
#   (Phase 8 CI-01) enforces zero literal matches of the qualified form
#   anywhere under internal/registry/; this comment deliberately
#   describes the identifier in prose rather than reproducing the
#   literal token, so this file itself passes the guard.
#
#   Failure mode: the keychain authenticator reads ~/.docker/config.json
#   which on a host that ran `docker login` with an empty username
#   emits "Authorization: Basic Og==" and breaks GHCR's anonymous
#   bearer flow. The Pitfall 2 regression guard in transport_test.go
#   asserts no inbound request ever carries Basic Og==.
